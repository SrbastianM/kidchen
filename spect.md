# Background
Dentro de la carpeta `concept` existe una imagen de referencia que define la identidad visual esperada del proyecto, incluyendo:
- Paleta de colores
- Estilo gráfico
- Composición visual
- Ambientación general

La implementación debe respetar y tomar como base dicho diseño.

# Objetivo
Crear una SPA (Single Page Application) utilizando React que tenga una estética psicodélica e inmersiva.

# Requerimientos funcionales
- Implementar un contador regresivo dinámico que finalice exactamente el:
  - Jueves 21 de mayo a las 7:11 PM
- El contador debe actualizarse en tiempo real (días, horas, minutos y segundos).

# Requerimientos visuales
- El fondo de la aplicación debe incluir múltiples ojos psicodélicos animados.
- Los ojos deben:
  - Parpadear de forma orgánica y aleatoria
  - Tener movimiento sutil
  - Integrarse visualmente con la estética general
- La ambientación debe sentirse:
  - Surrealista
  - Psicodélica
  - Hipnótica
  - Moderna y artística

# Diseño y estilo
- Utilizar exclusivamente la paleta de colores definida en la imagen dentro de `concept`.
- Mantener coherencia visual con la referencia.
- Priorizar una experiencia visual impactante y fluida.
- Agregar efectos visuales modernos como:
  - Glow
  - Blur
  - Gradientes animados
  - Noise sutil
  - Transiciones suaves

# Requerimientos técnicos
- Utilizar React.
- Se permite instalar e implementar las librerías necesarias para lograr animaciones avanzadas y una experiencia visual de alta calidad.
- Preferiblemente utilizar:
  - Framer Motion para animaciones
  - GSAP si es necesario
  - CSS moderno o TailwindCSS
- La SPA debe ser responsive.
- El rendimiento visual debe mantenerse fluido en desktop y mobile.

# Resultado esperado
Una landing page inmersiva y psicodélica con:
- Un countdown central llamativo
- Fondo animado con ojos parpadeando
- Animaciones fluidas
- Diseño visual inspirado completamente en la referencia de `concept`
- Apariencia moderna, artística y cinematográfica
